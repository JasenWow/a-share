# 自选股分析指南

## 什么是自选股分析？

自选股分析是 Big-A 系统的一项功能，让你对自己的股票池进行全面打分和趋势分析。它整合了 Kronos（时间序列预测）和 LightGBM（因子模型）两个模型，帮助你：

- 了解每只股票当前的多空信号
- 追踪股票过去一段时间的打分走势
- 模拟基于模型的推荐持仓
- 通过 AI 定性分析深入理解每只股票

想象你有一个私人量化团队，24小时不间断地监控你的自选股，告诉你每只股票现在怎么看、短期趋势如何、应该分配多少仓位。

---

## 功能一览

### 多模型打分

系统同时运行两个顶级量化模型：

| 模型 | 类型 | 特点 |
|------|------|------|
| Kronos | Transformer 时间序列 | 擅长捕捉短期波动趋势 |
| LightGBM | 因子模型 | 擅长多因子综合评分 |

两个模型的打分经过归一化后取平均，得出综合得分（范围 -1 到 1）。

### 趋势分析

除了当前打分，系统还能生成滚动历史打分：

- Kronos 滚动打分：每个交易日重新计算，生成 10 日趋势线
- LightGBM 滚动打分：展示过去 N 个交易日的打分变化

趋势图能帮你识别：
- 哪只股票正在转强
- 哪只股票开始疲软
- 是否有背离信号（模型打架）

### 持仓模拟

基于综合打分，系统会：

1. 选取打分最高的 5 只股票
2. 等权分配仓位（单只最高 25%）
3. 剩余资金保留为现金
4. 显示每个股票的信号（强烈看涨 / 看涨 / 看跌 / 强烈看跌）

### AI 定性分析（可选）

开启后，系统会调用多 Agent 工作流对每只股票进行深度分析：

- 📊 技术分析 Agent：识别图形模式、趋势强度
- 💰 估值分析 Agent：PE、PB、DCF 等估值指标
- 🏦 巴菲特 Agent：护城河、盈利能力评估
- ⚠️ 风控 Agent：系统性风险、流动性风险
- 📋 投资组合 Agent：与自选股其他股票的关联

## 命令行接口

### report - 完整报告

生成包含所有内容的完整分析报告。

```bash
python big-a/scripts/watchlist_report.py report [选项]
```

**常用选项：**

| 选项 | 默认值 | 说明 |
|------|--------|------|
| `-w, --watchlist` | `configs/watchlist.yaml` | 自选股配置文件路径 |
| `-a, --account` | `1000000` | 模拟持仓资金（元） |
| `-m, --model` | `kronos` | 使用模型：`all`、`kronos`、`lightgbm` |
| `-l, --lookback` | `5` | 历史趋势天数 |
| `-o, --output` | 空 | 保存报告到文件 |
| `-q, --qualitative` | False | 启用 AI 定性分析 |
| `--no-portfolio` | False | 不显示模拟持仓 |
| `--skip-trend` | False | 跳过滚动打分（大幅加速） |

**示例：**

```bash
# 基本用法（使用默认配置）
python big-a/scripts/watchlist_report.py report

# 使用 LightGBM 模型
python big-a/scripts/watchlist_report.py report -m lightgbm

# 同时使用两个模型
python big-a/scripts/watchlist_report.py report -m all

# 修改模拟资金
python big-a/scripts/watchlist_report.py report -a 500000

# 查看 10 日趋势（默认 5 日）
python big-a/scripts/watchlist_report.py report -l 10

# 跳过滚动计算（如果 Kronos 太慢）
python big-a/scripts/watchlist_report.py report --skip-trend

# 启用 AI 定性分析
python big-a/scripts/watchlist_report.py report -q

# 保存报告到文件
python big-a/scripts/watchlist_report.py report -o ~/Desktop/watchlist_report.txt
```

---

### score - 快速打分

只输出模型打分，不包含趋势分析、市场数据和持仓模拟。适合快速查看当前信号。

```bash
python big-a/scripts/watchlist_report.py score [选项]
```

**常用选项：**

| 选项 | 默认值 | 说明 |
|------|--------|------|
| `-w, --watchlist` | `configs/watchlist.yaml` | 自选股配置文件路径 |
| `-m, --model` | `kronos` | 使用模型：`all`、`kronos`、`lightgbm` |

**示例：**

```bash
# 快速打分
python big-a/scripts/watchlist_report.py score

# 使用两个模型
python big-a/scripts/watchlist_report.py score -m all
```

## 配置文件格式

自选股配置使用 YAML 格式，每行格式为：

```yaml
股票代码: 股票名称  # 可选备注
```

股票代码使用 Qlib 格式：`SH` 开头表示上海，`SZ` 开头表示深圳。

**示例配置（`configs/watchlist.yaml`）：**

```yaml
# 自选股列表 - 手动维护
# 格式: stock_code: 可选备注
# 股票代码使用 Qlib 格式: SH600000 / SZ000001
watchlist:
  SZ001309: 德明利
  SZ300476: 胜宏科技
  SH688256: 寒武纪
```

## 报告各部分解读

### [1] 模型打分

| 列 | 说明 |
|----|------|
| 代码 | Qlib 格式的股票代码 |
| 名称 | 配置文件中指定的名称 |
| Kronos | Kronos 模型预测的收益百分比 |
| LightGBM | LightGBM 模型的原始打分 |
| 信号 | 综合信号（🟢 看涨 / 🔴 看跌） |

**信号判断标准：**

| 综合得分 | 信号 | 含义 |
|----------|------|------|
| > 0.5 | 强烈看涨 | 两个模型都强烈看多 |
| > 0 | 看涨 | 略微看多 |
| > -0.5 | 看跌 | 略微看空 |
| <= -0.5 | 强烈看跌 | 两个模型都强烈看空 |

### [2] 10日分数走势

Kronos 和 LightGBM 各自的历史打分表格。绿色表示正收益，红色表示负收益。

帮你判断：
- 股票是否持续强势
- 是否出现拐点信号
- 两个模型是否一致

### [3] 近10日行情

每只股票最近 10 个交易日的 OHLCV 数据，帮助你结合价格走势理解模型打分。

### [4] 模拟持仓

基于模型打分推荐的模拟持仓：

- 选取打分最高的 5 只股票
- 等权分配，单只最高 25%
- 显示每只股票的入场信号和预估价格
- 现金仓位帮助你了解总体的多空态度

**注意：** 这只是模型打分生成的模拟持仓，不构成投资建议。

### [总结]

一行总结：
- 看涨股票数量 / 总股票数
- 平均综合得分
- 最强和最弱的股票

## 常见问题

### Q: 报告生成很慢怎么办？

A: 可以用 `--skip-trend` 跳过滚动打分，只保留当前打分：

```bash
python big-a/scripts/watchlist_report.py report --skip-trend
```

这会把时间从几分钟缩短到几十秒。

### Q: Kronos 模型加载很慢？

A: Kronos 首次加载需要下载模型（约 2GB），后续会缓存到本地。如果长时间没用，再次使用可能需要几分钟重新加载。

### Q: 两个模型打分方向不一致怎么办？

A: 这种情况说明市场存在分歧。建议：
1. 查看趋势图，判断哪个方向是主流
2. 启用 AI 定性分析，获取更多视角
3. 降低分歧股票的仓位权重

### Q: 如何只用一个模型？

A: 使用 `-m` 选项指定模型：

```bash
python big-a/scripts/watchlist_report.py report -m kronos    # 只用 Kronos
python big-a/scripts/watchlist_report.py report -m lightgbm # 只用 LightGBM
```

### Q: 提示"文件不存在"怎么办？

A: 检查配置文件路径是否正确：

```bash
# 使用绝对路径
python big-a/scripts/watchlist_report.py report -w /full/path/to/watchlist.yaml

# 使用相对路径（从 big-a 目录）
cd big-a
python scripts/watchlist_report.py report -w configs/watchlist.yaml
```

## 总结

自选股分析是你口袋里的量化助手：

| 功能 | 适用场景 |
|------|----------|
| `report -m all` | 全面分析，需要趋势、持仓、AI 分析 |
| `report --skip-trend` | 快速当前打分，不需要趋势 |
| `report -m kronos` | 只关心时序模型 Kronos |
| `score` | 最快速度看当前信号 |

记住：
- 第一次运行 Kronos 需要下载模型，稍等片刻
- 趋势分析可选，跳过可以大幅加速
- AI 定性分析需要 API Key，但能提供更深洞察
- 模拟持仓只是模型打分结果，不构成投资建议
